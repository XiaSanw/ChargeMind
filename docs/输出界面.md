# ChargeMind 报告输出界面设计

> **定位**：可视化驾驶舱 + 精简文字，不是一段长 Markdown  
> **数据来源**：后端 LLM 输出结构化 JSON → 前端渲染图表  
> **核心原则（嘉老师）**：体检报告式，越丰富越好，但信息以图表呈现，文字仅做点睛

```
## 输出要求

你必须输出一个严格的 JSON 对象（不要 Markdown，不要代码块标记），结构如下：

{
  "dashboard": {
    "headline": "一句话痛点诊断，必须量化差距",
    "overall_score": 0-100的整数,
    "radar": {
      "维度名": { "score": 0-100, "comment": "一句话诊断" }
    },
    "scoring_logic": "评分计算逻辑简述"
  },
  "kpi_cards": [
    { "label": "...", "value": "...", "trend": "up/down/flat", "benchmark": "...", "detail": "..." }
  ],
  "benchmark": {
    "labels": ["你的场站", "对标1", ...至少4个],
    "metrics": [
      { "key": "...", "name": "...", "unit": "...", "values": [数值, ...] }
    ]
  },
  "trend_projection": {
    "months": [0, 3, 6, 9, 12],
    "scenarios": [
      { "name": "保守/基准/乐观", "values": [数值, ...], "description": "..." }
    ]
  },
  "paths": [
    {
      "title": "...",
      "category": "成本优化/效率提升/博弈调价/资产盘活",
      "annual_gain": 万元数值,
      "probability": 0-1,
      "effort": "low/medium/high",
      "source": "[算法预测]/[知识库类比]/[行业规律推断]",
      "detail": "一句话说明逻辑"
    }
  ],
  "detail_text": "Markdown详细分析（唯一文字区）"
}

## 约束

1. **六维雷达**：固定六个维度——位置价值、硬件配置、运营效率、收益能力、竞争格局、增长潜力。每项给出 0-100 评分和一句话诊断。
2. **KPI 卡片**：固定四张——预测利用率、年收益预估、日均充电量、高峰时段。每张包含趋势方向。
3. **标杆对比**：至少包含 4 个对标对象（含区域均值），至少 3 个指标维度。
4. **趋势推演**：三情景（保守/基准/乐观）× 五个时间点（0/3/6/9/12 月）。
5. **提升路径**：至少 3 条，按 effort 排序（low → medium → high），每条标注来源。
6. **量化优先**：所有收益差距写具体数字，禁止"有较大提升空间"。
7. **聚焦直接回报**：不考虑人员成本、运维成本。
8. **detail_text 精简**：作为唯一文字区也控制在 500 字以内，点到为止。

## 六维度评分逻辑参考

- **位置价值**：周边 POI 密度 + 交通便利度 + 人口覆盖
- **硬件配置**：功率匹配度 + 快慢充比例 + 桩数适配
- **运营效率**：利用率 + 日均充电量 + 峰谷均衡度
- **收益能力**：年利润 + 度电毛利 + 投资回报周期
- **竞争格局**：周边竞品密度 + 价格竞争优势 + 差异化程度
- **增长潜力**：低谷利用空间 + 引流可能性 + 政策红利
```

### 4.4 核心组件实现要点

**RadarChart.tsx** — 六维雷达图：
```tsx
import { RadarChart, PolarGrid, PolarAngleAxis, PolarRadiusAxis, Radar } from 'recharts';

interface Props {
  data: RadarData;
  overallScore: number;
}

export function StationRadarChart({ data, overallScore }: Props) {
  const chartData = Object.entries(data).map(([key, val]) => ({
    dimension: key,
    score: val.score,
    fullMark: 100,
  }));

  return (
    <div className="relative w-full h-80">
      <RadarChart data={chartData} cx="50%" cy="50%" outerRadius="70%">
        <PolarGrid stroke="var(--border)" />
        <PolarAngleAxis dataKey="dimension" tick={{ fill: 'var(--foreground)', fontSize: 12 }} />
        <PolarRadiusAxis domain={[0, 100]} tick={false} axisLine={false} />
        <Radar dataKey="score" stroke="#3b82f6" fill="#3b82f6" fillOpacity={0.2} />
      </RadarChart>
      {/* 中心综合评分 */}
      <div className="absolute inset-0 flex items-center justify-center">
        <span className="text-3xl font-bold">{overallScore}</span>
      </div>
    </div>
  );
}
```

**KPICard.tsx** — 单张 KPI 卡片：
```tsx
import { TrendingUp, TrendingDown, Minus } from 'lucide-react';
import { Card } from '@/components/ui/card';

interface Props {
  card: KPICard;
}

const trendIcon = {
  up: TrendingUp,
  down: TrendingDown,
  flat: Minus,
};
const trendColor = { up: 'text-green-400', down: 'text-red-400', flat: 'text-yellow-400' };

export function KPICard({ card }: Props) {
  const Icon = trendIcon[card.trend];

  return (
    <Card className="p-4 flex flex-col gap-2 min-w-[160px]">
      <span className="text-sm text-muted-foreground">{card.label}</span>
      <span className="text-2xl font-bold">{card.value}</span>
      <span className={`text-xs flex items-center gap-1 ${trendColor[card.trend]}`}>
        <Icon size={14} /> {card.detail}
      </span>
      <span className="text-xs text-muted-foreground">{card.benchmark}</span>
    </Card>
  );
}
```

**BenchmarkChart.tsx** — 标杆对比柱状图（含指标切换）：
```tsx
import { BarChart, Bar, XAxis, YAxis, CartesianGrid, Tooltip, Legend, Cell, ResponsiveContainer } from 'recharts';

interface Props {
  benchmark: Benchmark;
  selectedMetric: string;
  onMetricChange: (key: string) => void;
}

export function BenchmarkChart({ benchmark, selectedMetric, onMetricChange }: Props) {
  const metric = benchmark.metrics.find(m => m.key === selectedMetric)!;
  const chartData = benchmark.labels.map((label, i) => ({
    name: label,
    value: metric.values[i],
    isUser: i === 0,
  }));

  return (
    <div>
      {/* Tab 切换 */}
      <div className="flex gap-2 mb-4">
        {benchmark.metrics.map(m => (
          <button
            key={m.key}
            onClick={() => onMetricChange(m.key)}
            className={`px-3 py-1 rounded text-sm ${m.key === selectedMetric ? 'bg-primary text-primary-foreground' : 'bg-muted'}`}
          >
            {m.name}
          </button>
        ))}
      </div>
      <ResponsiveContainer width="100%" height={300}>
        <BarChart data={chartData}>
        <CartesianGrid strokeDasharray="3 3" stroke="var(--border)" />
        <XAxis dataKey="name" tick={{ fill: 'var(--foreground)', fontSize: 12 }} />
        <YAxis tick={{ fill: 'var(--foreground)', fontSize: 12 }} />
        <Tooltip />
        <Bar dataKey="value" name={metric.name} unit={metric.unit}>
          {chartData.map((entry, i) => (
            <Cell key={i} fill={entry.isUser ? 'var(--primary)' : 'var(--muted-foreground)'} />
          ))}
        </Bar>
        </BarChart>
      </ResponsiveContainer>
    </div>
  );
}
```

**TrendChart.tsx** — 趋势推演折线图：
```tsx
import { LineChart, Line, XAxis, YAxis, CartesianGrid, Tooltip, Legend, ResponsiveContainer } from 'recharts';

const SCENARIO_COLORS = {
  '保守': '#f59e0b',
  '基准': '#3b82f6',
  '乐观': '#10b981',
};

export function TrendChart({ data }: { data: TrendProjection }) {
  const chartData = data.months.map((month, i) => {
    const point: Record<string, number | string> = { month: `${month}月` };
    data.scenarios.forEach(s => { point[s.name] = s.values[i]; });
    return point;
  });

  return (
    <ResponsiveContainer width="100%" height={280}>
      <LineChart data={chartData}>
      <CartesianGrid strokeDasharray="3 3" stroke="var(--border)" />
      <XAxis dataKey="month" tick={{ fill: 'var(--foreground)' }} />
      <YAxis tick={{ fill: 'var(--foreground)' }} unit="万" />
      <Tooltip formatter={(v: number) => `${v} 万元`} />
      <Legend />
      {data.scenarios.map(s => (
        <Line
          key={s.name}
          dataKey={s.name}
          stroke={SCENARIO_COLORS[s.name as keyof typeof SCENARIO_COLORS]}
          strokeWidth={2}
          dot={{ r: 4 }}
        />
      ))}
      </LineChart>
    </ResponsiveContainer>
  );
}
```

```tsx
export function ReportPage() {
```
